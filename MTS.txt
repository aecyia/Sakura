Tired of the blue UI ?

Want a light pink one ?

This project aims to recolour the UI, redesign icons, and more ♡

There will be many updates, and previous versions can be found [u][url=https://github.com/aecyia/Sakura/releases]here[/url][/u] ♡

It is mainly based on my personal preferences, but suggestions are welcome ♡

The colours may appear different based on your computer’s brightness and colour settings ♡

[u][i][b]Installation[/b][/i][/u]
[list][*] Download. Extract the file.
[*] Move to "Documents\Electronic Arts\The Sims 3\Mods\"
[*] Can be placed in Overrides OR Packages[/list]

[b][i][u]Conflicts[/u][/i][/b]
[list][*] Only use one recolour mod at a time
[*] Any other mod that changes the UI image assets
[*] If you have other mod packages that changes the assets, prioritize via Resource.cfg in your Mods folder[/list]
